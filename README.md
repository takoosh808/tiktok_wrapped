# TikTok Wrapped

TikTok Wrapped is a creator analytics startup that helps marketers, creators, and agencies turn raw TikTok hashtag data into clear, decision-ready insights.

Our product makes trend analysis simple: upload hashtag performance data, filter by year, and instantly surface top-performing topics and momentum shifts. Instead of guessing what might trend next, teams can make strategy decisions backed by data.

## Why We Exist

Short-form content moves fast, and trend cycles are short. Most teams lose time stitching together spreadsheets and dashboards before they can act.

TikTok Wrapped was built to answer one core question quickly:

What should we create next to stay relevant and grow?

## Product Snapshot

TikTok Wrapped currently includes:

- A modern analytics dashboard built with Next.js and TypeScript
- Upload flow for TikTok hashtag datasets
- Year-based filtering for historical trend exploration
- Core KPI cards and trend visualizations
- Top hashtag ranking views for fast content planning
- API routes for analytics and year metadata

## Who It Is For

- Creator economy startups
- Social media agencies
- Brand marketing teams
- Independent creators and strategists

## Startup Vision

We are building TikTok Wrapped into the operating system for short-form trend intelligence:

- Real-time trend alerts
- Vertical-specific benchmarks (beauty, finance, gaming, etc.)
- Competitive tracking across creators and campaigns
- AI-assisted content idea generation grounded in live trend signals

## Tech Stack

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS + shadcn/ui components
- API routes for analytics data services

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open `http://localhost:3000` in your browser.
